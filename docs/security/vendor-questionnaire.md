# Vendor Security Questionnaire

This will be the canonical answer set for district/vendor security questionnaires.

## Scope

- Product: InventoryK12
- Data: Student PII, staff PII, parent/guardian PII, asset inventory metadata

## MVP Questionnaire Checklist (Questions)

- Data types collected and stored (student, staff, parent/guardian PII; asset metadata)?
- Data storage locations and environments (dev, staging, prod)?
- Encryption at rest and in transit (what is used, where, and for which data)?
- Access control model (roles, least privilege, admin capabilities)?
- Authentication methods supported today and planned (MVP vs future SSO/MFA)?
- Audit logging coverage (asset assignments, admin actions, data access)?
- Data retention and deletion policy (timelines, triggers, export support)?
- Incident response process (contact, timelines, escalation)?
- Subprocessors and third-party services used (current and planned)?
- Backup and disaster recovery approach (RPO/RTO targets if defined)?
- Security testing practices (dependency scanning, code review, pen tests)?
- Compliance targets (FERPA, COPPA, Texas requirements; future SOC 2/ISO)?
