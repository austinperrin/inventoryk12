# Sprint Documentation

This directory stores sprint-specific planning and review records.

## Naming

- Use `sprint-000-<name>.md` for kickoff/foundation work.
- Use `sprint-###.md` for subsequent sprints (`###` zero-padded).

## Current Sprint Docs

- `sprint-000-foundation-alignment.md`

## Usage

- Record sprint goals, scope, risks, and completion evidence.
- Keep roadmap milestone statuses aligned with sprint outcomes.
